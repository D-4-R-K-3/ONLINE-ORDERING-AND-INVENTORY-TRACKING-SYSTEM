## Installation & Setup

### Prerequisites
- PHP 8.0+
- MySQL/MariaDB 5.7+
- Apache with mod_rewrite
- Composer

### Installation Steps

1. **Database Setup**
   - Open phpMyAdmin
   - Create a new database (e.g., `ordering_system`)
   - Import `database_setup.sql`
   - Sample data is included (users, products, categories)

2. **Configuration**
   - Edit `app/Config/Database.php` with your database credentials
   - Set environment: `ENVIRONMENT = development` (or `production`)

3. **File Permissions**
   - Set `writable/` directory permissions to 0755
   - Set `public/uploads/` directory permissions to 0755

4. **Start Application**
   - Run: `php spark serve`
   - Access: `http://localhost:8080`

## API Endpoints

### Products
- GET `/api/products` - List all products
- GET `/api/products/{id}` - Get product details
- POST `/api/products` - Create product
- PUT `/api/products/{id}` - Update product
- DELETE `/api/products/{id}` - Delete product

### Orders
- GET `/api/orders` - List orders (authenticated)
- GET `/api/orders/{id}` - Get order details (authenticated)
- POST `/api/orders` - Create order (authenticated)
- PUT `/api/orders/{id}` - Update order (authenticated)

### Categories
- GET `/api/categories` - List all categories
- GET `/api/categories/{id}` - Get category

### Inventory
- GET `/api/inventory` - List all inventory
- GET `/api/inventory/{id}` - Get inventory item
- PUT `/api/inventory/{id}` - Update inventory
