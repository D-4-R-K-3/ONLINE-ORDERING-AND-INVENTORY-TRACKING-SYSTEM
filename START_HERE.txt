🚨 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

DATABASE ERROR
   → Check app/Config/Database.php credentials
   → Verify MySQL is running
   → Ensure database exists
   → Verify user permissions

VIEW NOT FOUND
   → Check file exists in app/Views/
   → Verify file naming (case-sensitive)
   → Check view path in controller

PERMISSION ISSUES
   → chmod 755 writable/
   → chmod 755 public/uploads/
   → Verify web server user permissions

SESSION ISSUES
   → Clear browser cookies
   → Check session storage in app/Config/App.php
   → Verify writable folder is accessible
